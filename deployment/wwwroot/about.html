The CoralMap is a website for map-based access to spatial data from Australian previous census and predictions to the future immigration patterns of Australia. It is a capstone project as part of the Master Degree in Information Technology specialised in Articial Intelligence at The University of Melbourne .

The CoralMap is a fully open architecture that is based on TerriaMap and TerriaJS. When you access data through it, which has been transformed and processed from its original source the government department Australian Bureau of Statistics who is the custodian of that data.

The CoralMap...

  • provides easy access to country of origin data at the Local Government Area from census 2006
  • Makes predictions about the country of origin for census 2021 with a high degree of precision
  • Predictions are made with machine learning algorithms using various data inputs such as income, job, industry, religion and cultural background

CoralMap Software

The NationalMap is Open Source software available as a Github project. The core platform, TerriaJS, was initially developed by Data61 for the NationalMap, but has subsequently been used for other projects. It has been made available to assist with the development of other spatial data viewing applications.

Additional Open Source Software Used

In developing the NationalMap, Data61 has used open source software and contributed back to it. NationalMap leverages these excellent open source libraries:

  • TerriaJS
  • Cesium
  • Leaflet

The complete list is available with the NationalMap source code
